# 🎬 Clone da Tela de Login da Netflix

Projeto desenvolvido em **React Native** com o objetivo de reproduzir a **tela de login da Netflix**.

## 📌 Sobre o projeto

Este projeto foi criado para praticar conceitos de desenvolvimento mobile utilizando React Native, com foco na construção da interface inspirada na página de login da Netflix.

## 🛠️ Tecnologias utilizadas

- React Native
- TypeScript
- Expo
